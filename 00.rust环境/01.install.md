1. ubuntu安装
   + `sudo apt remove rustc cargo`   # 只删 apt 安装的，不影响后续 rustup
   + `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh` # 官方一键脚本安装 rustup
   + 选 1）default (stable) 即可；脚本会把~/.cargo/bin 加到 PATH，重新登录终端或执行：
   + `source ~/.cargo/env`
   + rustc --version   # 1.82.0 或更高
   + cargo --version   # 同上
   + rustup --version  # 1.27.x
2. windows直接去官网下载`rustup-init.exe`执行
3. 如果安装不对就去设置rustup镜像
