rustup —— ** toolchain 管理器 **（生态的“大管家”）
| 功能                        | 例子                                                                                |
| --------------------------- | ----------------------------------------------------------------------------------- |
| 安装/卸载 工具链            | `rustup toolchain install 1.70.0` / `rustup toolchain uninstall nightly-2024-01-01` |
| 切换默认工具链              | `rustup default stable` / `rustup default nightly`                                  |
| 给项目指定专用工具链        | `rustup override set 1.70.0`（会在项目根写 `rust-toolchain.toml`）                  |
| 更新自己 & 全部链           | `rustup update`                                                                     |
| 装组件（rust-src、clippy…） | `rustup component add clippy rust-src rust-analyzer`                                |
| 装目标平台交叉编译          | `rustup target add aarch64-unknown-linux-gnu`                                       |
