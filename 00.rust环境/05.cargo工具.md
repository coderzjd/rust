cargo —— 项目前端 & 构建系统
| 子命令                                     | 说明                                 |
| --------------------------------------- | ---------------------------------- |
| `cargo new/bin/lib`                     | 创建项目脚手架                            |
| `cargo check`                           | 最快语法/类型检查（不生成二进制）                  |
| `cargo build` / `cargo build --release` | Debug / Release 编译                 |
| `cargo run`                             | 编译并立即运行                            |
| `cargo test`                            | 跑单元 + 集成测试                         |
| `cargo bench`                           | 运行基准测试（需 nightly 或 stable+bencher） |
| `cargo clippy`                          | 运行官方 linter（插件式 lint 集合）           |
| `cargo fmt`                             | 自动格式化（调用 rustfmt）                  |
| `cargo doc --open`                      | 生成并打开项目文档                          |
| `cargo publish`                         | 把 crate 上传到 crates.io              |
| `cargo install xxx`                     | 从 crates.io 安装二进制到 `~/.cargo/bin`  |
